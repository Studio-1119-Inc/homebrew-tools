# Studio 1119 Homebrew Tap

Homebrew formulas for Studio 1119 developer tools.

## Install

```bash
brew tap Studio-1119-Inc/tools
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `md-to-gdoc` | Convert Markdown files to Google Docs with cross-file link rewriting and upsert |

```bash
brew install Studio-1119-Inc/tools/md-to-gdoc
```
